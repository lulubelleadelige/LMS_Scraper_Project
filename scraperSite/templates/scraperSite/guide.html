Moodle Security Guide

Course Scanning

The system will automatically scan active courses and courses are going to be active in 4 weeks every Saturday at 1:00 AM.

Dashboard Reports

The dashboard visualization will be automatically sent to the admin email every Sunday. To view the logs, go to Dashboard -> Select Period -> View Scanned Log.

Log history

Scanned activities and login/logout log is recorded and can be found in logs folder inside project.

File scanning

It will take more time while running scanning files. This feature only enable when folder is mounted in this demo version.

AI loading

This system is using AI for URLs scanning. Therefore, the scanning speed might be slow on courses that have various URLs and files
